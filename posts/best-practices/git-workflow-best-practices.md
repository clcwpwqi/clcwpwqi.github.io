# Git 工作流与最佳实践

Git 是目前最流行的分布式版本控制系统。本文介绍 Git 的高级用法和团队协作最佳实践。

## 分支策略

### Git Flow

经典的分支模型，适合发布周期明确的项目。

### GitHub Flow

简化的工作流，适合持续部署。

## 提交规范

```bash
git commit -m "feat(auth): add OAuth2 login support"
git commit -m "fix(api): resolve null pointer exception"
```

## 总结

良好的 Git 工作流能够提高团队协作效率。
