# VS Code 效率提升指南

Visual Studio Code 是目前最流行的代码编辑器之一。

## 快捷键

| 快捷键 | 功能 |
|--------|------|
| Cmd/Ctrl + P | 快速打开文件 |
| Cmd/Ctrl + Shift + P | 命令面板 |

## 推荐插件

- ESLint
- Prettier
- GitLens

## 总结

熟练掌握 VS Code 的快捷键和插件，能够显著提升开发效率。
